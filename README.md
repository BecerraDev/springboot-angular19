# Prueba Técnica - Desarrollador Java

## Descripción
Proyecto full-stack con Java + Spring Boot (backend), Angular (frontend) y autenticación con Keycloak.

## Tecnologías
- Java 17
- Spring Boot 3
- Angular 19
- PostgreSQL (opcional)
- Keycloak 26
- Docker / Docker Compose

1. Clona el repositorio
2. Ejecuta:
